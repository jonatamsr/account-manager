# Account Manager Project

A project designed for monetary accounts management.

## Requirements
- php8.2^
- composer2.5.8^

## Setup

1. Install dependencies: `composer install`
2. Start serving: `php -S localhost:8000 -t public`

## Run Tests

`php vendor/phpunit/phpunit/phpunit`
